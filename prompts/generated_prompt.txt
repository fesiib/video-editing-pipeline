Identify from the command:

1. Video segment reference: timecodes, transcript mentions, actions, or visual elements.
2. Frame region reference: specific locations, objects, or areas of interest.
3. Editing operation (text, image, shape, blur, cut, crop, zoom) and its parameters if mentioned.

Classify each video segment reference as:
1. "position": timecodes, timesegovsegments, abstract position (e.g. "intro") 
2. "transcript": transcript references
3. "action": actions in the video
4. "visual": visual elements in the video
5. "context": high-level segment description

Output Format (JSON-like):
{
    "temporal": [],
    "temporal_labels": [],
    "spatial": [],
    "edit_operation": [],
    "parameters": {
        "text": [],
        "image": [],
        "shape": [],
        "blur": [],
        "crop": [],
        "zoom": [],
    },
}

Examples:
Command: "Zoom into the pan around 1:31 when he says "flip chicken after 6 min""
Output:
{
    "temporal": ["1:31", ""flip chicken after 6 min""],
    "temporal_labels": ["position", "transcript"],
    "spatial": ["pan"],
    "edit": ["zoom"],
    "parameters": {},
}

Command: "Add “Cooking lemon herb chicken...” in bold, arial font in intro. Blur lightly that gradually disappears once he starts speaking."
Output:
{
    "temporal": ["intro", "once he starts speaking"],
    "temporal_labels": ["position", "transcript"],
    "spatial": ["over the video"],
    "edit": ["text", "blur"],
    "parameters": {
        "text": ["“Cooking lemon herb chicken...”.", "bold", "arial"],
        "blur": ["light", "gradually disappears"],
    },
}
